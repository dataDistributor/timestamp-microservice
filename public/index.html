Timestamp Microservice

Welcome to the Timestamp Microservice. Use the following endpoints:

  • /api - Returns the current timestamp.
  • /api/2023-01-01 - Returns the timestamp for a valid date.
  • /api/1450137600 - Returns the timestamp for a UNIX time.

If you provide an invalid date, you'll receive an error message:

{ "error": "Invalid Date" }
