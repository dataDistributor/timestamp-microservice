Timestamp Microservice

Enter a date or timestamp in the URL after /api/.

Examples:

  • /api/2025-01-01
  • /api/1450137600
  • /api (current date)

The response will be a JSON object:

{
  "unix": 1450137600000,
  "utc": "Fri, 15 Dec 2015 00:00:00 GMT"
}
